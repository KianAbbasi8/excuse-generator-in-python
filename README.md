# Python Excuse Generator

## Description
A fun Python program that generates random excuses for work, school, or daily life by combining a subject, a reason, and a time. This mini-project was built to practice Python fundamentals including list operations, random selection, and string concatenation. Created as part of my learning journey with **@Codanics**.

## Features
- Randomly selects from pre-defined lists:
  - Subjects (who is involved)
  - Reasons (what happened)
  - Times (when it occurred)
- Combines them into a complete, humorous excuse
- Simple command-line interface
- Each run produces a unique excuse

## How to Run
1. Ensure Python 3 is installed on your system
2. Clone or download this repository
